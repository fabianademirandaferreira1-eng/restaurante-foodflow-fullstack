# 🍔 FoodFlow - Gestão de Restaurante Full Stack

Sistema desenvolvido para controle de fluxo de caixa e gestão de pedidos de um restaurante em tempo real.

## 🚀 Tecnologias Utilizadas
* **Front-end:** HTML5, CSS3, JavaScript (Vanilla)
* **Back-end:** Node.js, Express
* **Banco de Dados:** MySQL
* **Ferramentas:** XAMPP, Git & GitHub

## 🛠️ Funcionalidades
- [x] Cadastro de pedidos via interface web.
- [x] Backend integrado com banco de dados relacional.
- [x] Dashboard de faturamento total automático.
- [x] API REST para comunicação entre Front e Back.
